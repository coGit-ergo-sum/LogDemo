Vi.Log4Vi Namespace

This namespace belongs to the 'general' namespace 'Vi' (a collection of company wide tools) but refers to an autonomous project which has the aim to provides a simple way to 'interact' with Log4Net. (A very powerfull tool, not so easy to configure)
